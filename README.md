🇮🇳 Bharat Explorer

Bharat Explorer is a travel discovery web application built using Django that allows users to explore tourist destinations across India state-wise. Users can view detailed information about places, explore images, check the best time to visit, and save destinations to a personal wishlist.

🗺️ Browse tourist destinations state-wise across India

⚙️ How to Run Locally
1️⃣ Clone the repository
git clone https://github.com/niteshkumar9368/bharat_explorer.git
cd bharat_explorer
2️⃣ Create a virtual environment
python -m venv venv

Activate it:

Windows

venv\Scripts\activate

Mac / Linux

source venv/bin/activate
3️⃣ Install dependencies
pip install -r requirements.txt
4️⃣ Apply database migrations
python manage.py migrate
5️⃣ Run the development server
python manage.py runserver

Now open:

http://127.0.0.1:8000
